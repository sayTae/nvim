# Essential Packages  
  
Please download the following package as required.  
  
> [vim-Plug] -> all plug-in  
> [node.js] -> coc  
> [ctag] -> tagbar  

## 1. vim-plug
Reference Repository
```
https://github.com/junegunn/vim-plug
```
## 2. node.js
Linux
```
sudo apt install node.js
```
Mac
```
brew install node
```
## 3. ctags
Linux
```
sudo apt install exuberant-ctags
```
Mac
```
brew install ctags
```
